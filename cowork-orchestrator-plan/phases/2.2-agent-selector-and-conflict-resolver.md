# 2.2 — AI Agent Selector & Conflict Resolver

```
Continuing "cowork-orchestrator". TaskDecomposer is done. Now implement agent selection and conflict resolution.

## 1. src/intelligence/agent-selector.ts — AI Agent Selection

Implement `AgentSelector` class:
- Constructor takes `Anthropic` client

- `selectAgent(task: Task, candidates: BaseAgent[]): Promise<AgentSelectionResult>`
  - Builds a prompt with: task requirements, and for each candidate: { id, name, role, capabilities, currentLoad, stats }
  - Uses tool_use to get structured response:
    { selectedAgentId: string, reasoning: string, confidence: number (0-1), alternativeId?: string }
  - If confidence < 0.5, set a `needsHumanReview: true` flag on the result

- `rankAgents(task: Task, candidates: BaseAgent[]): Promise<RankedAgent[]>`
  - Returns all candidates ranked: { agentId, score (0-100), reasoning }

- `suggestTeam(workflow: Workflow, availableAgents: BaseAgent[]): Promise<TeamAssignment[]>`
  - Given a full workflow, suggest which agent handles each node
  - TeamAssignment: { nodeId, agentId, reasoning }
  - Claude considers: load balancing, specialization, minimizing context switches

- LRU cache for recent selections (cache key = hash of task.name + agent IDs, TTL = 5 minutes, max 100 entries)

Define types:
  interface AgentSelectionResult { selectedAgentId: string; reasoning: string; confidence: number; alternativeId?: string; needsHumanReview?: boolean; }
  interface RankedAgent { agentId: string; score: number; reasoning: string; }
  interface TeamAssignment { nodeId: string; agentId: string; reasoning: string; }

## 2. src/intelligence/conflict-resolver.ts — AI Conflict Resolution

Implement `ConflictResolver` class:
- Constructor takes `Anthropic` client and optional `MessageBus`

- `detectConflicts(outputs: Map<string, TaskResult>): Promise<Conflict[]>`
  - Takes outputs from multiple agents, sends to Claude asking: "Do any of these outputs contradict, overlap, or conflict?"
  - Returns array of Conflict: { type: 'contradiction' | 'overlap' | 'inconsistency', agentIds: string[], description: string, severity: 'low' | 'medium' | 'high', suggestedResolution: string }

- `resolve(conflict: Conflict, agentOutputs: Map<string, TaskResult>): Promise<Resolution>`
  - Claude picks a strategy: MERGE | PREFER | RETRY | ESCALATE
  - Resolution: { strategy, resolvedOutput?: Record<string,unknown>, reasoning: string, retryAgentIds?: string[] }

- `preventConflict(tasks: Task[]): Promise<ConflictPrevention>`
  - Before parallel execution, ask Claude if tasks might conflict
  - ConflictPrevention: { riskLevel: 'none' | 'low' | 'medium' | 'high', guardrails: string[] }

Define the Conflict, Resolution, ConflictPrevention types in this file or types.ts.

Run `npx tsc --noEmit` after each file.
```
